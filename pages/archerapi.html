ArcherAPI: Data of dlsu, for dlsu.

What data does this provide?

Right now, the API provides the data of all classes in DLSU. (Limited to: Term 3, A.Y. 2023-2024)


However, I have planned for it in the future to support data on professors, more description on rooms, buildings, and other things.


How was this achieved?

I scraped the list of all valid course codes in DLSU through the pre-enlistment facility, searching each course code from A-Z. After, I fed all those courses into the MLS through web scraping, saving the class schedules available of the term.


I found out the following information about Term 3, A.Y. 2023-2024:

  • 4100~ total classes active
  • 12,822 total courses all time in DLSU (including old/deprecated courses)
  • 1,200~ total courses with classes active in term 3.
